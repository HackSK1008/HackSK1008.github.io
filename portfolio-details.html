Project Overview

FitMantra is a social-impact project designed to make quality fitness guidance accessible using computer vision, gamification and IoT.

Key Features
  • Real-Time Posture Correction — reduces injuries using pose estimation.
  • Gamified Motivation — points, leaderboards and streaks to boost adherence.
  • Personalized Plans — analytics-driven plans tailored per user.
  • IoT Integration — optional sensors for richer tracking.
Technologies

Python, OpenCV, MediaPipe, TensorFlow, Flask, Flutter, IoT

Outcome & Recognition

Cost-effective solution for gym-goers; awarded 1st Prize — Healthcare Domain at an international hackathon hosted by PICT Pune.

Visit live website

Technical Summary

FitMantra combines optimized pose-estimation with lightweight models to run efficiently on mobile devices and edge units. It focuses on accuracy, low-latency feedback and privacy-preserving data handling.

Architecture
  1. Video capture → Pose Estimation → Posture Analysis
  2. Local feedback loop (visual + audio), gamification engine, server-side analytics
  3. Optional IoT sensor aggregation for advanced metrics
Results
  • Improved form correction accuracy in user tests.
  • Increased average workout adherence in pilot users.
